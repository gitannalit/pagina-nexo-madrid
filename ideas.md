# Ideas de Diseño - Nexo Madrid Landing Page

## Enfoque Elegido: "Luxury Dark Gold — Poder y Exclusividad"

**Design Movement:** Neo-Luxury Premium Dark — inspirado en la identidad visual de Nexo Dubai, combinando la sofisticación de una marca de lujo con la urgencia de conversión de alta performance.

**Core Principles:**
1. Negro profundo como base absoluta — transmite poder, exclusividad y seriedad financiera
2. Dorado como color de acción y valor — cada CTA, métrica y elemento destacado usa oro
3. Tipografía de contraste fuerte — títulos en peso extrabold para impacto, cuerpo legible
4. Urgencia visual constante — contadores, barras de progreso, plazas limitadas siempre visibles

**Color Philosophy:**
- Background: #0a0a0a (negro casi absoluto)
- Cards/Sections: #111111 / #161616
- Gold Primary: #C9A84C (dorado cálido, no amarillo)
- Gold Bright: #D4AF37 (para CTAs y highlights)
- Text: #FFFFFF / #E5E5E5 / #AAAAAA
- Borders: rgba(201, 168, 76, 0.3)

**Layout Paradigm:**
- Sección hero asimétrica: texto a la izquierda, carrusel a la derecha
- Secciones alternadas con fondo ligeramente diferente para crear ritmo visual
- Sticky bar inferior con info del evento + CTA siempre visible
- Ticker animado en la parte superior

**Signature Elements:**
1. Números grandes en dorado (40-50%, 200+, 10X) como anclas visuales
2. Badges "EXCLUSIVO" en esquina superior de tarjetas de beneficios
3. Barra de progreso de plazas ocupadas con urgencia visual

**Interaction Philosophy:**
- Hover en tarjetas: sutil brillo dorado en el borde
- CTAs con efecto shimmer/brillo al hover
- Acordeón FAQ con transición suave
- Carrusel automático con transición fade

**Animation:**
- Ticker de texto continuo en la parte superior
- Fade-in suave en secciones al hacer scroll
- Número animado (count-up) para métricas clave
- Carrusel hero con transición fade cada 4 segundos

**Typography System:**
- Títulos: Playfair Display (serif elegante) para H1/H2 principales
- Subtítulos y UI: Montserrat (sans-serif premium) para H3, labels, CTAs
- Cuerpo: Montserrat Regular para texto descriptivo
- Números/Métricas: Montserrat ExtraBold para impacto máximo
